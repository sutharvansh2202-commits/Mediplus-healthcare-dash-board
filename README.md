-Project Overview

This project presents an interactive Power BI dashboard developed using the MediPlus dataset.
The goal of this project is to analyze healthcare-related data and generate meaningful insights through visualization and data modeling.

The dashboard enables users to monitor key performance indicators, trends, and patterns in medical or healthcare-related data efficiently

-Objectives

Analyze healthcare data effectively

Visualize KPIs using interactive charts and graphs

Identify patterns and trends in patient or medical data

Provide decision-support insights

-Tools & Technologies Used

Microsoft Power BI

Data Modeling (Relationships, Measures)

DAX (Data Analysis Expressions)

Data Cleaning & Transformation

Interactive Visualizations
